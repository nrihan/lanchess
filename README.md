## Descrição
<p>Apresentaremos uma implementação de um jogo de xadrez que conecta cliente-servidor para que dois jogadores possam jogar um contra o outro via Rede. O intuito desse projeto é mostrar a conexão de dois clientes ao mesmo servidor para que duas máquinas consigam se comunicar.
</p>
